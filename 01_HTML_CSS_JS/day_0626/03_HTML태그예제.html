HTML 태그 예제

<blockquote>와 <q>

이것은 블럭 단위 인용입니다. 여러 줄의 인용문에 사용됩니다.

짧은 인용은 인라인으로 이렇게 사용합니다.

<time>

문서가 작성된 날짜는 2025년 06월 26일입니다.

<b>와 <strong>

굵은 텍스트는 시각적인 강조만 의미하며, strong은 의미상 강조를 나타냅니다.

<i>와 <em>

이텔릭 텍스트는 스타일 강조이고, em은 의미상 입니다.

<mark>

중요한 정보를 하이라이트 할 수 있습니다.

<span>

이 부분은 스타일링만을 위한 span 태그입니다.

<s>

취소선은 더 이상 유효하지 않은 정보를 나타낼 때 사용합니다.

<code>와 <pre>

코드 블럭은 <code> 태그로 나타냅니다. 


                function hello() {
                    console.log("Hello, World!")
                }

   (non-breaking space)
