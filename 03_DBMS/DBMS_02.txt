RENAME TABLE 기존이름 TO 새이름; - 테이블 이름 바꾸기
ALTER TABLE 기존이름 RENAME TO 새이름; - 테이블 이름 바꾸기

DROP TABLE 테이블이름; - 테이블 삭제
DROP DATABASE 데이터베이스이름; - 데이터베이스 삭제

ALTER TABLE 테이블이름 CHANGE 기본이름 새이름 ...; - 컬럼 이름 / 속성 바꾸기

DELETE FROM 테이블이름 WHERE 조건; - 테이블에서 조건에 만족하는 행 삭제.
ㄴ 예제 : DELETE FROM employee WHERE name = '손흥민';

SELECT 조건 소수점자리연산(연산, 소수점자리) AS 연산이 들어간 행의 이름 FROM 테이블이름; - 연산이 들어간 테이블 확인 및 이름 만들기
ㄴ 예제 : SELECT name, salary, salary * 1.5 AS 퇴직금 FROM employee;
ㄴ 소수점자리연산 : ROUND / CEIL / FLOOR == 소수점 반올림/내림 / 소수점 없애고 올리기 / 소수점 없애고 내리기


** 인덱스 활용 **
ALTER TABLE 테이블이름 ADD INDEX 인덱스이름 (조건); : 인덱스 만들기
ㄴ 예제 : ALTER TABLE employee ADD INDEX idx_name (name);


** 계정 관련 **
CREATE USER '계정명'@'호스트' IDENTIFIED BY '비밀번호'; - 계정생성 (호스트는 로컬인지 외부인지)

GRANT 권한 ON 데이터베이스.테이블 TO '계정명'@'호스트'; - 권한 (SELECT, DROP, CREATE 등)을 계정에 부여
GRANT ALL PRIVILEGES *.* TO '계정명'@'호스트' WITH GRANT OPTION; - root와 거의 똑같은 권한을 가진 계정 생성

:: 호스트 종류 ::
localhost		= 로컬 컴퓨터에서 접속
%		= 모든 IP / 호스트 허용
127.0.0.1		= 로컬 IP / localhost와 유사하지만 TCP 소켓으로 연결
192.168.0.%	= 192.168.0.으로 시작하는 내부망 IP 대역만 허용
example.com	= 특정 도메인에서만 접속 가능
host1		= DNS에 등록된 호스트 이름
