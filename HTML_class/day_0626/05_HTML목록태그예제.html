2. 용어 정리 (정의 목록)

HyperText Markup Language의 약자로 웹 페이지의 뼈대를 구성하는 언어입니다.

Cascading Style Sheets의 약자로 HTML 요소에 디자인을 적용합니다.
JavaScript
